2025_02_Peace Hunt Koshi
【概要】
このアプリは、ユーザーが合志市の観光地を訪れることで合志市に関連する写真のピースを収集するアプリである。

【起動】 アプリ一覧から「2025_02_Piece Hunt Koshi」をタップする。

【使い方】
1.アプリを起動するとスタート画面が起動し、位置情報の使用の許可を行うかどうかの通知が出ます。許可を押すと「スタート」ボタンが押せるようになるので、「スタート」ボタンを押してください。
2.スタートボタンを押すとマップ画面に移行します。マップ画面ではマップ内に現在地が青い丸と矢印で、ランドマークの位置と範囲が赤いピンと円で表示されます。そのランドマークが目指す場所です。ランドマークのピンをタップすると、その場所の情報を見ることができます。
3.右上にあるリストボタンを押すと、ランドマーク一覧が表示され、ランドマークの名前と現在地からランドマークまでの距離が一覧表示される。
4.「パズル」ボタンを押すとパズル画面に移行し、現在収集しているパズルの収集具合が表示され、残りの枚数が表示されます。「マップに戻る」ボタンでマップ画面に戻ります。
5.「コレクション」画面では今まで集めたパズルの一覧を見ることができる。パズルのサムネイルを押すことでそのパズル画面に移行できます。
5.設定されたランドマーク内（赤い円）に入ると「〇〇のエリアに入りました！」と通知が表示され「ピースをゲット！！」ボタンが押せるようになります。
（評価時は、左上のテスト位置ボタンから熊本高専をランドマークにできるので、左上の「テスト位置追加」ボタンを押してランドマークに入った際の画面に遷移してください）
6.「ピースをゲット！！」ボタンを押すとピース入手画面に移行し、場所の名称とスマホを振ろう！！と表示されます。仰ぐようにスマホを振ったり、横に振るとピース入手画面に移行します。
7.「ピースNo.〇〇を手に入れた！」と表示され現在収集中のパズルのランダムなピースを入手できます。「マップに戻る」ボタンでマップ画面に戻ります。
8.ピースをゲットすると、その場所は、クールタイム中となり、「ピースをゲット」ボタンを押せなくなり、24時間待つ必要があります。残り時間は、右上のリストボタンから見ることができます。(テスト位置の場合は表示されません)
9.ピースをゲットした場所で24時間経過すると、再度ピースをゲットすることができます。(評価時には、左上のクールタイムリセットボタンを押すことで何度もピースをゲットできるのでご利用ください)
10.ピースを9個集め、パズルを完成させると、完成の演出を楽しむことができます。
11.パズルを完成させると、次のパズルが始まります。(熊本高専→教室)
【一時停止】
端末の「ホームボタン」で、今の状態のままアプリを一時停止してホーム画面になる。
再度アプリを起動すると、一時停止前の状態からスタートする。
