【概要】 
このアプリは、ユーザーが合志市の観光地を訪れることで合志市に関連する写真のピースを収集するアプリである。 

【起動】
アプリ一覧から「2025_02_Piece Hunt Koshi」をタップする。

【使い方】 

1.アプリを起動するとスタート画面が起動し、位置情報の使用の許可を行うかどうかのダイアログ(Allow 2025_02_Piece Hunt Koshi to access this device's location?)が表示される。許可(ALLOW)を押すと画面下部の「スタート」ボタンが押せるようになる。 
「スタート」ボタンが押せない場合は、スタート画面左上の「アプリ権限の設定」ボタンを押し、「Permissions」を選択した後、「Location」の右側のボタンを押す。ボタンが緑になれば、位置情報の仕様を許可したことになるので、もう一度スタート画面に戻れば「スタート」ボタンが押せるようになる。
二回目以降のアプリの使用で位置情報の利用許可がない場合は、「アプリを継続するためには位置情報の取得が必要です」というダイアログが表示される。「OK」を選択すれば位置情報の使用の許可を行うかどうかのダイアログが表示される。

2.スタートボタンを押すとマップ画面に移行する。マップ画面ではマップ内に現在地が青い矢印で、ランドマークの位置と範囲が赤いピンと円で表示されている。画面下部の「パズル」ボタンを押すとパズル画面に移行し、「コレクション」ボタンを押すとコレクション画面に移行する。マップ画面右上のリストボタンを押すとランドマーク一覧が表示され、ランドマークの名前と現在地からランドマークまでの距離が一覧表示される。 

3.パズル画面では現在収集しているパズルの収集具合が表示され、残りの枚数が表示される。「マップに戻る」ボタンでマップ画面に戻る。 

4.コレクション画面では今まで集めたパズルの一覧を見ることができる。パズルのサムネイルを押すことでそのパズル画面に移行できる。 

5.ランドマーク内に入ると「〇〇のエリアに入りました！」と通知が表示され「ピースをゲット！！」ボタンが押せるようになる。 

6.「ピースをゲット！！」ボタンを押すとピース入手画面に移行し、スマホを振ろう！！と表示される。スマホを振るとピース入手画面に移行し、「ピースNo.〇〇を手に入れた！」と表示され現在収集中のパズルのランダムなピースを入手する。「マップに戻る」ボタンでマップ画面に戻る。 

 【一時停止】 
端末の「ホームボタン」で、今の状態のままアプリを一時停止してホーム画面になる。 
再度アプリを起動すると、一時停止前の状態からスタートする。
